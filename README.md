# 🔥 Fire Alarm System using Arduino

## Description
This project simulates a Fire Alarm System using Arduino in Tinkercad. It detects high temperature and activates an LED and buzzer.

## Components
- Arduino Uno
- Temperature Sensor
- LED
- Buzzer
- Resistors
- Breadboard

## How It Works
- Reads the temperature sensor.
- Compares the value with a threshold.
- Turns on the buzzer and LED if the threshold is exceeded.
- Turns them off when the temperature returns to normal.
